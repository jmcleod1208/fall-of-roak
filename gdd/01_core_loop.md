# 01 Core Loop

## Purpose

Define what the player repeatedly does, why it is satisfying, and how each loop feeds the next.

## Status

**Design State:** Unlocked template.

**Last Locked Update:** None.

## Core Loop Layers

### Moment-to-Moment Loop

Define what the player does every few seconds during combat or preparation.

### Encounter Loop

Define what the player does before, during, and after a fight.

### Session Loop

Define the normal 10-to-30-minute play pattern.

### Meta Progression Loop

Define what persists between sessions and why the player returns.

### Narrative Loop

Define how story progress ties into combat, travel, party growth, or region advancement.

## Rewards and Friction

### Primary Rewards

List the rewards that create forward momentum.

### Failure States

Define what losing means mechanically.

### Recovery

Define how the player recovers from failure without breaking tension.
